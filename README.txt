== Description ==

Contributor Plugin that show Post Contributors List after the content.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Contributor'
3. Activate Buddy Views from your Plugins page.

= From WordPress.org =

1. Download Buddy Views.
2. Upload the 'contributor' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Contributor from your Plugins page.

== Frequently Asked Questions ==

= Will it just show contributor users list?  =

Yes, It will also show contributor profile picture with respective author page link.

= can we assign all user role as contributor?  =

No, only admin can select user which user role is admin, author and editor.
